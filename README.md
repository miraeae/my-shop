# 🛒 My Shop

JSON 파일을 활용해 데이터를 바인딩하여 간단한 장바구니 쇼핑몰 사이트를 구현한 프로젝트입니다. 비동기 처리와 localStorage 실습을 위해 제작하였으며, 현재 완성도를 높이기 위해 계속 작업을 진행하고 있습니다.

- **제작기간** 2025.04.24 ~ 진행중
- **사용언어** HTML, CSS, JavaScript, JSON
- **라이브러리** Swiper
- **유형** PC 적응형
